# Figure 2 and Figure S2-S4 Code

This folder contains cleaned, panel-oriented code for Figure 2 and Figure S2-S4.

## Sample Labels

Sample names used in figures and output files are defined in `config/figure2_inputs.csv` and applied by `R/00_setup.R`.

## Scripts By Panel

- `config/panel_map.csv`: detailed panel-to-script mapping for Figure 2 and every Figure S2/S3/S4 subpanel.
- `config/supplementary_panel_parameters.csv`: supplement-specific samples, region groups, marker lists, and reused code.
- `config/sample_specific_parameters.csv`: explicit per-sample clustering and cell2location settings.
- `config/region_marker_sets.csv`: region-label marker genes used for Figure 2/S3/S4 annotations.
- `config/random_celltype_parameters.csv`: random B/T cell-type assignment rules used for Figure 2J/K and supplement panels.
- `config/cellchat_parameters.csv`: CellChat pathway lists, grouping columns, and pathway-specific panels.
- `python/01_cell2location_mapping.py`: cell2location mapping workflow used before Figure 2 region/cell-type plots.
- `R/01_prepare_cell2location_inputs.R`: exports Seurat counts and metadata for cell2location.
- `R/figure2A_region_annotation_umap_spatial.R`: Figure 2A, region annotation on UMAP and tissue coordinates.
- `R/figure2B_celltype_composition_by_region.R`: Figure 2B, cell-type composition by annotated region.
- `R/figure2C_marker_violin.R`: Figure 2C, marker-gene violin plot supporting region annotation.
- `R/figure2D_E_F_celltype_abundance_violin.R`: Figure 2D-F, region-specific cell-type abundance comparisons.
- `R/figure2G_J_shannon_diversity_by_region.R`: Figure 2G/J, IGH/TRB clonotype Shannon diversity.
- `R/figure2H_shm_rate_by_region.R`: Figure 2H, IGH SHM rate by region.
- `R/figure2I_K_correlation_heatmaps.R`: Figure 2I/K, cell-type abundance versus clonotype metric correlations.
- `R/figure2J_K_random_celltype_assignment.R`: Figure 2J/K, random B/T cell-type assignment, cell-type Shannon diversity, and B-T correlation heatmap.
- `R/figure2L_cellchat_interaction_strength.R`: Figure 2L, CellChat interaction strength heatmap.
- `R/figure2M_cellchat_pathway_networks.R`: Figure 2M, pathway-specific CellChat networks and ligand-receptor contribution.
- `R/figureS2_supplementary_cellchat_correlation.R`: Figure S2 supplementary correlations and communication panels.
- `R/figureS3_LN2_region_analysis.R`: Figure S3 supplementary panels for LN2.
- `R/figureS4_Tonsil1_region_analysis.R`: Figure S4 supplementary panels for Tonsil1.

## Supplement-Specific Notes

Figure S2-S4 reuse the main Figure 2 helper functions where the logic is identical. The supplement scripts still define their own panel-level parameters:

- Figure S2:
  - S2A/S2B reuse the Figure 2I/K Spearman correlation heatmap function for IGH/TRB.
  - S2C/S2D use the CellChat object generated by `R/figure2L_cellchat_interaction_strength.R`.
  - S2E/S2F reuse the Figure 2M pathway plotting function for BAFF, MHC-II, and CXCL.
- Figure S3:
  - Uses the LN2 sample.
  - S3-specific region labels, marker genes, and abundance-comparison cell types are declared in `R/figureS3_LN2_region_analysis.R`.
- Figure S4:
  - Uses the Tonsil1 object.
  - S4A QC plotting and S4D marker dotplot are supplement-specific.
  - S4B/C/E-G/F-H reuse Figure 2 region, composition, Shannon, and correlation helper functions with S4-specific region labels.

## Explicit Parameters

The source scripts are intentionally helper-based, but the public repository keeps the biological and statistical parameters visible:

- Per-sample Seurat settings are in `config/sample_specific_parameters.csv`.
- Region marker genes for each sample are in `config/region_marker_sets.csv`.
- Random cell-type assignment uses IGH for B-cell clonotypes, TRB for T-cell clonotypes, and cell2location q05 abundance columns as spot-level probabilities; see `config/random_celltype_parameters.csv`.
- CellChat pathway panels use the pathway lists and grouping columns in `config/cellchat_parameters.csv`.

## Running

Run scripts from the repository root after filling in the file paths in `config/figure2_inputs.csv`.
For example:

```r
source("analysis/figure2/R/figure2B_celltype_composition_by_region.R")
```

Outputs are written to `analysis/figure2/outputs/`.
